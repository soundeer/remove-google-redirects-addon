remove google redirects addon

Firefox addon to simply remove tracking code/redirect from Google search results. Works with firefox 9.x or superior.

Download this project as a .zip file Download this project as a tar.gz file

It has been built using addon-sdk 1.9. You may download the approved version from Firefox Add-ons website or get the latest one from this repository: click to install.
